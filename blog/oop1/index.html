Before discussing about Object Oriented Programming We have to know what is Procedural Programming. Procedural Programming (For example: C) focuses on functions. But what is a function ?

First of all a program is a set of instructions for carrying out a specific task.

But a function is also a set of instructions for carrying out a specific task.

So what's the difference ?

Difference between A Program and A function is that the function is smaller unit or a block of code which can be reused within in A program.

Now that we have learned what Procedural Programming is, let us discuss what is Object Oriented Programming (OOP).

In a sentence OOP is a programming paradigm that focuses on objects (Remember, Procedural Programming focuses on functions !!!). There are 4 pillars or core concepts of OOP. They are Abstraction, Inheritance, Encapsulation, Polymorphism. These concepts directly or indirectly related to Classes and Objects. So firstly we have to learn these two.

So what are objects ?

Objects are real life entity which has some properties and behaviors. These properties called variables and behaviors are called functions !

So basically by with Object Oriented Programming it is so much easier to represent a real life entity. For example: A man named "John" is an object, whose color is white and height is 1.75 meter. Here color and height are properties of John with values "white" and "1.75 meter". Let's say he can cook. So cooking is his one behavior. So we can see that by OOP it is easier to represent everything !

Now if we want to define Classes, In a sentence, Classes are blueprints from where we can create that type of objects.

In our previous example, we said "A man named 'John' ". Consider 'Man' here is a Class. 'Man' has color, height, age etc. 'Man' can have different behaviors. Example above 'John' is a man who can 'Cook'. It's John's behavior.

Don't be confused! We can create many objects from a Class. Let's say we have another Object of 'Man' class named 'Bob'. Bob's color is black. Height is 1.75 meter. He can ride bikes. So 'Riding' is Bob's behavior.
